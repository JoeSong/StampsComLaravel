StampsComLaravel
==============
A PHP wrapper for the Stamps.com API, for use with Lumen 5.3 or higher.

To use run `composer require JoeSong/StampsComLaravel`

Setup your API credentials in Lumen's `.env` file:
```
STAMPS_INTEGRATION_ID=xxxx
STAMPS_USERNAME=xxx
STAMPS_PASSWORD=xxx
```
